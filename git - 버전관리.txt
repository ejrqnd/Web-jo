git - 버전관리






git add "파일이름" -> 해당 파일만 추가
git add . -> 현재 모든 폴더의 모든 변경점을 추가

버전기록 : git commit -m "메세지"

어떤 버전이 기록되어있는지 확인 : git log
되돌리기 : git reset 옵셥 커밋ID

옵션 :
--SOFT -> 커밋 내용은 사라지는데 내용을 임시 저장 상태로 되돌림.
--MOXED -> 임시저장도 취소한다.작업공간에는 남겨둠.
--HARD -> 임시저장도 취소, 작업공간에서도 삭제.
HEAD : 최신 버전을 의미한다.		(이전버전으로 가고싶으면 HEAD~1)

원격저장소에 올리기전 반드시 커밋(commit)이 완료된 상태여야한다. 				git push -u origin master	원격저장소 올리기 (origin은 임의이름)

	add.		commit		push
폴더 -> stage -> 버전기록 -> 원격 저장소에 저장
